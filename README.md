# 🇳🇱 IPTV & IP Radio – Netherlands

A simple collection of **Dutch IPTV and Internet Radio playlists** in `.m3u` format.

These playlists are intended for use with IPTV players such as VLC, Kodi, IPTV Smarters, TiviMate, and similar apps.

> ⚠️ Note: Streams are intended for viewers/listeners in the Netherlands.

# ▶️ How to Use

2. Open your IPTV player.
3. Add a **new playlist via URL**.
4. Paste the link and load the playlist.
